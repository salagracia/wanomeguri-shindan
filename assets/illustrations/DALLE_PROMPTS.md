# 🎯 サラさんへのお願い

## 進め方

1. **時間が取れる時に** ChatGPT Plus を開く（一気に15枚作ると30-45分くらい）
2. 上記プロンプトを **1個ずつコピー** → ChatGPT に貼り付け → 送信
3. 生成された画像を **指定ファイル名** でダウンロード
4. ダウンロードした画像を `D:\CCBooster\dna-shindan-v2\assets\illustrations\` に保存
5. 15枚全部揃ったら、**ウィルに「画像できた」と教えてください**

## ウィルが次にやること

サラさんから「画像できた」の合図をもらったら：
1. PDF生成コードに画像埋め込みを追加（30分）
2. ローカルテスト
3. コミット＆プッシュ
4. Render自動再デプロイ
5. **次回診断から、画像付きの美しいPDFが届く** 🌹

## もし気に入らない画像があったら

- 同じプロンプトを ChatGPT に **「もう少し○○な感じで」** と追加指示で再生成
- それでも合わなければ、ウィルに「これだけ作り直したい」と教えてください
- プロンプトを調整して再提示します

## 一気に15枚は大変なら

- まず **#01 序章 と #02 第1章** だけ作って試してみる
- ウィルが先にPDF埋め込みを実装
- 「あ、こういう感じか」が分かったら、残り13枚を一気に

このペースで進めるとサラさんの負担が最小化されます。
