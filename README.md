# 45歳からの 信頼される男 診断

YouTube「**信頼される男の流儀**」公式診断アプリ。

仕事はそれなりにやってきたのに結果が頭打ちで、家でも妻に必要とされていない気がする——
そんな45歳以上の男性に向けて、**「仕事での強みが家庭で裏目に出ている構造」**を診断で明らかにし、
最も近い人に信頼され、結果として仕事も伸びる男に変えていくための個人設計図を生成します。

## 仕組み

- 24問の選択式診断（仕事軸／家庭軸／価値観軸）
- 8占術（数秘・西洋占星術・九星気学・四柱推命・動物キャラ・算命学・帝王学・姓名判断）
- 4タイプ判定（A統率／B職人／C調整／D挑戦）＋ 隠れタイプ
- 自由記述4問（仕事の没頭体験／信念／妻への本音／空回りの正体）
- 個別PDF（約24ページ）を自動生成し、ユーザーのメールアドレス宛に送付

## 章構成

| 章 | 内容 |
|---|---|
| 序章 | あなたが頭打ちなのは、能力のせいじゃない |
| 第1章 | あなたの本質（占術データ） |
| 第2章 | あなたの信頼される男タイプ（仕事の強みを認める） |
| 第3章 | あなたの中に眠るもう一つの顔（隠れタイプ） |
| 第4章 | 人生の追い風とブレーキ |
| 第5章 | これからの立て直しシナリオ |
| **第6章** | **最も近い人との関係（仕事の強みが家庭で裏目に出る構造）** |
| 第7章 | 数字に宿る、人生のテーマ |
| 第8章 | 名前に宿る力 |
| 第9章 | あなたの言葉が映す本質 |
| 巻末 | 兄貴分からの手紙 → 60分・個別相談（3,000円）誘導 |

## 技術スタック

- Streamlit
- ReportLab（PDF生成）
- skyfield / lunar-python（占術計算）
- Resend（メール送信）

## ローカル実行

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```
